# Nobel Prize Data Analysis

This project explores Nobel Prize winner data using Python.

## Objective
The goal of this project is to analyze Nobel Prize data and investigate patterns in category distribution, gender representation, and lifespan.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Questions
- Which Nobel Prize categories are most represented?
- How has gender distribution changed over time?
- Do Nobel laureates in Physics die younger than laureates in other categories?

## Key Finding
An initial hypothesis suggested that physicists might die younger than laureates in other categories. However, the analysis did not support this assumption.

## Repository Structure
- `data/` → dataset
- `notebooks/` → analysis notebook
- `images/` → exported charts
- `requirements.txt` → dependencies

## How to Run
1. Install dependencies from `requirements.txt`
2. Open the notebook in Jupyter Notebook
3. Run the cells in order
